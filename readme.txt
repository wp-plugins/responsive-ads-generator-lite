== Description ==

The Responsive Ads Generator Lite Plugin allows you to insert ads anywhere in your post just by using a simple shortcode. A great solution for displaying multiple ads in a simple grid layout. Responsive design looks great on mobile devices. On page refresh ads will rotate to keep your displays looking fresh. 


= RESPONSIVE ADS GENERATOR HAS A NUMBER OF EXCELLENT FEATURES: =

1. Easy-to-use interface for adding and generating ads
2. Random display of ads on page refresh.
3. Dynamic shortcode for easy use on posts and pages.
4. Select which ads are generated.
5. Preview generated ads in admin.

== Installation ==

1. Extract and upload the `responsive-ads-generator` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in the WordPress admin panel
3. Navigate to 'Dashboard > Responsive Ads Generator Lite' to create ads.

== Upgrade Notice ==

If you have customized the code and want to keep it, save before you update.

= Manual update =

1. Delete the plugin `eye-on-demand` folder under the `/wp-content/plugins/` directory
2. Upload the last version and activate it

Just use the Wordpress automatic plugin update system
